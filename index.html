Para que el tracker guarde las tareas en la nube, seguí estos pasos una sola vez:

  1. Abrí Google Sheets → creá una planilla nueva → llamala Tracker 1A
  2. En esa planilla: Extensiones → Apps Script
  3. Borrá todo, pegá el contenido del archivo tracker-script.gs y guardá (💾)
  4. Clic en Implementar → Nueva implementación → tipo: Aplicación web
  5. Ejecutar como: Yo · Acceso: Cualquier persona → Implementar
